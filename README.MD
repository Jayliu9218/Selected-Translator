# 划词翻译器

## 如何使用：

1. 编辑配置文件，包括翻译模式等；
2. 运行main.exe；
3. 选择要翻译的单词或句段； 
4. 将鼠标悬停至合适位置，键盘按下“ctrl + C”； 
5. 查阅翻译完毕，关闭翻译窗口
